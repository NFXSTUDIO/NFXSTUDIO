Procédure d'installation : 

- Décompresser le fichier zip dans le dossier voulue.
- Ouvrez windows PowerShell ou equivalent selon votre machine
- taper la commande suivante : cd chemin_acces_dossier/WATER --> chemin_acces_dossier étant le dossier ou se situe les fichiers après décompression.
- taper ensuite cet commande : py -m http.server

Une fois cela effectuer ouvrez votre navigateur internet est tapé : http://localhost:8000/water.html

Vous arrivez alors sur la page du navigateur des jeux effectué lors de CS485 !

Installation procedure:

- Unzip the zip file in the desired folder.
- Open Windows PowerShell or equivalent depending on your machine 
- type the following command: cd folder_access_path/WATER --> path access_folder being the folder where the files are located after unzip.
- then type this command: py -m http.server

Once this is done, open your internet browser and type: http://localhost:8000/water.html

You will then arrive on the browser page of the games played during CS485!
